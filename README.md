# 二手工程机械出口门户网站

## 项目概述
这是一个专业的二手工程机械出口B2B门户网站，支持多语言展示、WhatsApp集成和在线询盘功能。

## 主要功能
- 🌍 多语言支持（英语/西班牙语）
- 📱 WhatsApp一键沟通集成
- 📝 在线留言系统
- 🔍 产品筛选和展示
- 📱 移动端响应式设计
- 🚀 SEO优化

## 技术栈
- Next.js 16.2.1 (App Router)
- TypeScript
- Tailwind CSS
- 简化多语言路由
- SQLite数据库（开发环境）

## 部署
项目已配置Vercel部署，支持自动构建和部署。

## 访问地址
- 英语版本: `/en`
- 西班牙语版本: `/es`
- 产品列表: `/products`
- 联系我们: `/contact`

## 特性
- 产品卡片展示WhatsApp快捷沟通
- 全局悬浮WhatsApp按钮
- 在线留言表单
- 产品筛选功能
- 响应式设计
